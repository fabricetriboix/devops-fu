date: 2018-06-14
categories: [development]
tags: []
---

Open Chromium, go to the website to test, then hit:

{% highlight shell %}
Ctrl+Shift+J
Ctrl+Shift+M
{% endhighlight %}

On Mac:

{% highlight shell %}
Command+Shift+J
Command+Shift+M
{% endhighlight %}

Then you can select your device, orientation, etc.
